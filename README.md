# mjcampagna.github.io

Personal site, published at **https://mjcampagna.github.io**.

Built with [Jekyll](https://jekyllrb.com/) and the [Minima](https://github.com/jekyll/minima) theme. Deployed automatically to GitHub Pages via GitHub Actions on push to `master`.

## Local development

Requires Ruby 3.3. If using rbenv:

```bash
rbenv install 3.3.0
```

Then:

```bash
bundle install
bundle exec jekyll serve
```

Open `http://localhost:4000`.
